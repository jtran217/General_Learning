# General_Learning

A personal engineering lab — built by hand, no AI-generated code.

---

## What this is

This repo is where I sharpen the fundamentals. Every line of code here is written by me, deliberately, to keep my muscle memory sharp and my critical thinking intact. No Copilot completions, no pasted AI output — just reading, thinking, and typing.

The goal is breadth with depth: working through real software concepts by actually implementing them, not just reading about them.

AI is welcome as a **teacher and guide** — for explaining concepts, answering questions, reviewing logic, and pointing me in the right direction. It is not here to write the code.

---

## What I'll cover

Anything really. Trying to better understand code I've seen from AI in other projects and such.


## Rules

1. **No AI-generated code.** I write it. I own it (EXCEPT FRONTEND, cause honestly I really dislike it.
2. **AI is a resource**, not a crutch — same as a textbook or a Stack Overflow thread.
3. **Working over polished.** A messy implementation I understand beats clean code I copied.
4. **Document the struggle.** If something was hard to figure out, I write it down.
5. **Build to understand**, not to ship.

---

## Why

The fastest way to forget something is to let a tool do it for you. This repo exists because I want to actually know what I'm doing — to be able to sit down without autocomplete and still build things. The goal isn't to avoid AI; it's to never *need* it to do the thinking for me.
